# 🚴 Bike Sharing Dashboard

## 📊 Project Overview
This project analyzes bike sharing data from 2011–2012 to understand usage patterns based on time, weather conditions, working days, and user types. The results are presented through an interactive Streamlit dashboard.

---

## 🚀 How to Run Dashboard

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run Streamlit app
streamlit run dashboard/dashboard.py

## 🔍 Key Insights
- Bike usage peaks in the morning and evening (commuting hours)
- Clear weather leads to the highest number of rentals
- Working days have higher usage compared to holidays
- Registered users dominate total bike usage
